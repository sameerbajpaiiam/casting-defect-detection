# Industrial Casting Defect Detection

**Deep Learning Image Classification System**

Built a deep learning-based image classification system to identify defective vs non-defective casting products using a Kaggle dataset.

## Key Highlights
- Benchmarked a **custom CNN** against **MobileNetV2** transfer-learning model
- Achieved significant improvement in classification performance
- Rigorous model evaluation using `sklearn.metrics` and Matplotlib
- Generated classification reports, loss/accuracy curves, and ensured robustness & generalizability

## Tech Stack
- Python
- TensorFlow / Keras
- OpenCV
- Scikit-learn
- Matplotlib
